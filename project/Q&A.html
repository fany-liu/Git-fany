首頁 / 常見問題
Q1 : 我該如何預訂？
A1:在預訂畫面上選擇出發日期和旅客數量之後，請點擊立即預訂。這將帶您到“預訂”頁面。填寫必要的資料後，您將被引導到付款畫面。付款完成後，您會立即收到一封電子郵件確認信。
Q2：如果我找不到適合自己旅行日期的旅行，或者未滿足特定旅行的最低客人人數怎麼辦？
A2：別擔心。通過電話（+ 886-2-1234567）或電子郵件與我們聯繫，我們將盡力讓您於您首選的旅行日期出發或與其他客人一起合團旅行。
Q3：我可以使用哪些付款方式支付旅行費用？
A3 : Visa / MasterCard / AE 信用卡
Q4：如果我選擇了錯誤的出發日期，我可以調整我的預訂嗎？
A4 : 如果客戶選擇了錯誤的出發日期，請至少在旅行前三天聯繫service@tibame.com。我們的客戶服務團隊將協助您更改旅行日期。日期更改不能在旅行日期後的3天內進行。
Q5：在哪裡可以獲得導遊或司機的聯繫資料？
A5 : 我們至少在旅行日期前一天為您提供司機或導遊的聯繫方式。請隨時確認您的電子信箱。
Q6：如何查看我的訂單狀態？
A6 : 您可以通過your tour會員帳戶查看訂單狀態。如果您忘記了帳戶，也可以通過電子郵件發送至service@tibame.com。我們的客戶服務團隊將幫助您確認訂單狀態。
